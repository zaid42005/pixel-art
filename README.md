
# pixel-art

> python pixel art converter

## Development setup

Clone this repo and install the following:
 ```
pip install pipenv
pipenv --python 3.11
pipenv install streamlit
pipenv install numpy
pipenv install opencv-python-headless
pipenv install pillow
pipenv shell
```
